# MVP Profile Mockup R1

## Scope
Mobile `/plot` profile behavior and expanded-profile information architecture.

## Collapsed Profile (Top Strip)
- Avatar (left)
- Username
- Notifications icon (top-right area)
- Options menu (`...`)

No additional fields in collapsed state.

## Expanded Profile Layout
Expanded profile replaces Plot body area and keeps player context available.

### A) Profile Header Block (Top of Expanded Panel)
- Larger avatar
- Username + handle
- Activity score
- Band membership (if applicable)
- Promoter status (if applicable)

### B) Collection Menu (Below Header)
Default tab: `Singles/Playlists`

Tab order:
1. `Singles/Playlists`
2. `Events`
3. `Merch`
4. `Saved Uprises`
5. `Saved Promos/Coupons`

## Tab Content Contracts

### 1) Singles/Playlists (Default)
- Saved singles list
- Playlist groupings
- Collection-first controls only

### 2) Events
- Calendar at top
- Collected fliers list below calendar

### 3) Merch
- Category shelves:
  - Posters
  - Shirts
  - Patches
  - Buttons
  - Special Items

### 4) Saved Uprises
- Saved/followed uprise list
- Scene identity and quick entry context

### 5) Saved Promos/Coupons
- Promo/coupon cards
- Expiration/state indicators

## Notes
- This document defines structure and content only (not final visual styling).
- Collapsed profile remains exactly: avatar, username, notifications, options menu.
