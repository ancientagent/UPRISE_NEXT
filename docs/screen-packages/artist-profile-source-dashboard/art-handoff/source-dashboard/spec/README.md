# Source Dashboard Art Handoff Specs

Status: active handoff notes
Package: `artist-profile-source-dashboard`
Screen/section: `source-dashboard`

This folder stores the creative brief and asset contract for Source Dashboard
visual handoff work.

These files connect visual direction back to the package Design Spec and Dev
Spec. They are not owner specs and do not create runtime authority. Durable
product/API/runtime rules stay in `docs/specs/**`, current runtime code, and the
screen-package Dev Spec.

Current files:

- `2026-07-06_source-dashboard-creative-brief-v01.md` - visual target,
  hierarchy, Product Design constraints, and do-not-build boundaries.
- `2026-07-06_source-dashboard-asset-contract-v01.md` - what can be stored
  later, naming, approvals, rights/privacy, and links back to implementation
  authority.
