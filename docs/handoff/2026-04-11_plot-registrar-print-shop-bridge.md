# 2026-04-11 Plot Registrar Print Shop Bridge

## Summary
- Plot now exposes `Open Print Shop` from the `Registrar Access` panel.
- The CTA appears only when the current account already has a source-facing creator lane:
  - active promoter capability, or
  - at least one linked Artist/Band source.

## Why
- Home/Plot is still the main participation shell.
- After event creation went live, the remaining friction was forcing eligible creators to leave Home and know the Print Shop route manually.
- This keeps creator access visible in the same civic/status area that already surfaces registrar state.

## Boundaries
- No new creator eligibility was added.
- This is only a route/entrypoint bridge.
- Listener accounts without a creator lane still only see `Open Registrar`.
