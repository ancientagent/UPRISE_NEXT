# 2026-04-25 - Player Pull-Down Profile Rule

## Purpose
Record the founder correction that the user's profile / collection workspace is opened by pulling down the player, not by navigating to a separate normal profile page.

## Locked Interaction Truth
- The player starts as the top-screen listening infrastructure.
- Pulling the player down opens the user's profile / collection workspace in-place.
- While the profile / collection workspace is open, the player relocates to the bottom of that workspace.
- If space allows, the bottom player may keep normal controls/status.
- If space is tight, the bottom player may shrink into a compact scrolling marquee for band / song title.
- While the profile / collection workspace is open, Plot tabs are replaced.
- Collapsing the profile restores the prior Plot context.
- This is not a generic social profile navigation pattern.
